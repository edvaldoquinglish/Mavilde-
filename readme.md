# 💚 Para Mavilde — Landing Page Romântica

> *"Se o amor fosse código, o teu seria open-source."*

---

## 📖 Sobre

Uma landing page romântica criada com todo o amor do mundo, feita especialmente para **Mavilde** — com poesias geek, efeitos visuais cósmicos e uma trilha sonora em dois movimentos.

---

## ✨ Funcionalidades

- **Tela de entrada** com efeito de terminal digitando
- **Partículas flutuantes** tipo estrelas no fundo
- **Player de música** que toca `musica1.mp3` e transita suavemente para `musica2.mp3`
- **Versos com efeito de digitação** no hero, letra por letra
- **Poemas geek** que aparecem ao rolar a página (scroll reveal)
- **Glitch animado** com o nome "MAVILDE" na transição musical
- **Versos flutuantes** que aparecem a cada 15–20 segundos
- **Chuva binária** na seção final
- **Nome final animado** com brilho esmeralda e pulso de coração

## 🎵 Como Adicionar as Músicas

Coloca os teus ficheiros de música na pasta raiz do projeto com os nomes exatos:

- `musica1.mp3` — toca desde o início
- `musica2.mp3` — inicia na seção de transição (quando o nome "MAVILDE" aparece no glitch)

> **Formatos suportados:** `.mp3`, `.ogg`, `.wav`  
> Se quiseres usar outros nomes, edita a variável `playlist` no `script.js`.

---

## 🚀 Como Publicar no GitHub Pages

1. Cria um repositório no GitHub (ex: `mavilde-amor`)
2. Faz upload de todos os ficheiros desta pasta
3. Vai em **Settings → Pages**
4. Em "Source", escolhe `main` e pasta `/root`
5. Clica em **Save**
6. O teu site estará em: `https://teu-usuario.github.io/mavilde-amor/`
